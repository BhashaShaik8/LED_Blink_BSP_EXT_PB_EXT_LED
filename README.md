# STM32H747I-DISCO BSP LED and Button Demo

## Hardware
- STM32H747I-DISCO Discovery board
- External push button on PJ3 (CN6 pin3)
- External LED with 260 ohm resistor on PJ4 (CN6 pin5)

## Features
- BSP onboard LED control (GREEN, ORANGE, RED, BLUE)
- Onboard wakeup button toggles external LED
- External push button toggles external LED
- On board wakeup button 

## IDE
- STM32CubeIDE
- Core: CM7 (Cortex-M7)

## How to use
1. Open project in STM32CubeIDE
2. Build CM7 project
3. Flash to board via ST-Link
